Chain Management

Monitoring#

Each Wasp node provides a Grafana dashboard where it's possible to visualize metrics gathered by the node. // TODO expand on this

The chain state can be viewed via the dashboard (by default on port :7000).

Setting up a client to receive published events can also be a good way to have insights on what's happening in the system.

Lastly, each Wasp node will produce a log file (wasp.log) where the behaviour of a node can be investigated.

Managing Chain Configuration and Validators#

Managing the chain configuration and committee of validators can be done by interacting with the Governance contract.
